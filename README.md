# TuTaEcos - Zalo Multi-Tool 🚀

**1-Click tool to instantly create multiple Zalo PC instances. (Công cụ 1 click để tạo hàng loạt bản Zalo PC tức thì.)**

Tool được xây dựng bằng file .bat nên cực kỳ sạch. Như mình chia sẻ trước đó, code mở, không giấu code mục đích để mọi người cùng trao đổi kiến thức, rất vui nếu tool hữu ích với các bạn.

## 📂 Các file trong Tool (Included Files)
* `TUTAECOS - ZALO MULTI-TOOL.bat`: File dùng để tạo và mở hàng loạt ứng dụng Zalo PC.
* `TUTAECOS - ZALO CLONE UNINSTALLER.bat`: File dùng để dọn dẹp và xóa các bản Zalo clone khi không còn sử dụng. *(Lưu ý: Thay thế bằng tên file thật của bạn)*

## 🚀 Hướng dẫn sử dụng (How to Use)

**1. Để tạo nhiều Zalo (To create instances):**
* Tải file `.bat` về máy tính.
* Click chuột phải vào file `TUTAECOS - ZALO MULTI-TOOL.bat` và chọn **Run as administrator**.
* Tận hưởng việc mở hàng loạt Zalo PC chỉ với vài thao tác!

**2. Để xóa các Zalo clone (To delete instances):**
* Đảm bảo bạn đã tắt hết các Zalo clone đang mở.
* Click chuột phải vào file xóa clone và chọn **Run as administrator**.
* Chờ tool tự động dọn dẹp sạch sẽ hệ thống của bạn.
